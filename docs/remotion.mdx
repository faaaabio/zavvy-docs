O Remotion permite criar vídeos programaticamente com React — ideal para automatizar a montagem dos vídeos com o personagem verde, legendas e motion design padronizados.

---

## O que é o Remotion

O Remotion é uma ferramenta que transforma código React em vídeo. Em vez de editar manualmente cada peça no Premiere ou CapCut, você define a composição uma vez e gera quantas variações quiser — mudando legenda, cena ou formato via parâmetros.

**Uso na Zavvy:** montar os reels com o vídeo gerado pelo Seedance + legenda + CTA final, no padrão 4:5.

---

## Fluxo de produção

<Steps>
  <Step title="Gerar o vídeo base">
    Use a API do Freepik Seedance para gerar o clipe com o personagem verde. Veja [Gerar imagem com IA](/docs/gerar-imagem-com-ia).
  </Step>
  <Step title="Montar no Remotion">
    Acesse o [Remotion CLI](http://remotion-cli.vercel.app/), carregue o vídeo e configure legenda, CTA e duração.
  </Step>
  <Step title="Exportar">
    Exporte no formato final — 4:5, 720p — pronto para publicar no Instagram ou TikTok.
  </Step>
</Steps>
