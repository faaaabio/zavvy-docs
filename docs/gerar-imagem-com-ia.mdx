O Freepik oferece acesso gratuito via API ao modelo **Seedance** — o mesmo disponível na plataforma web. É com ele que geramos os vídeos do personagem verde.

**Configuração recomendada:** 720p · 4:5 · 5s · 24fps

<Card title="Interface web para geração" icon="globe" href="https://freepik-api-eight.vercel.app/" target="_blank">
  Use a interface visual para gerar vídeos sem precisar da API diretamente.
</Card>

---

## Autenticação

Todas as requisições precisam da chave de API no header.

```bash
x-freepik-api-key: SUA_CHAVE_AQUI
```

Para obter sua chave: **freepik.com → conta → API Keys**. O plano gratuito inclui gerações mensais sem custo.

---

## Gerar um vídeo

### Endpoint

```
POST https://api.freepik.com/v1/ai/video/seedance/text-to-video
```

### Parâmetros recomendados

```json
{
  "prompt": "Seu prompt aqui",
  "duration": 5,
  "resolution": "720p",
  "ratio": "4:5"
}
```

| Campo | Valor padrão Zavvy | Opções disponíveis |
|-------|-------------------|-------------------|
| `duration` | `5` | `5` ou `10` segundos |
| `resolution` | `"720p"` | `"480p"`, `"720p"`, `"1080p"` |
| `ratio` | `"4:5"` | `"16:9"`, `"9:16"`, `"4:5"`, `"1:1"`, `"21:9"` |

## Fluxo de geração

A geração é **assíncrona**: você envia o prompt, recebe um `task_id` e consulta o status até o vídeo ficar pronto.

```
POST /text-to-video  →  task_id  →  GET /tasks/{id}  →  vídeo pronto
```

### 1. Enviar o prompt

```bash
curl -X POST "https://api.freepik.com/v1/ai/video/seedance/text-to-video" \
  -H "Content-Type: application/json" \
  -H "x-freepik-api-key: SUA_CHAVE_AQUI" \
  -d '{
    "prompt": "SEU_PROMPT_AQUI",
    "duration": 5,
    "resolution": "720p",
    "ratio": "4:5"
  }'
```

Resposta:

```json
{ "data": { "task_id": "abc123-..." } }
```

### 2. Verificar o status

```bash
curl "https://api.freepik.com/v1/ai/video/seedance/tasks/abc123-..." \
  -H "x-freepik-api-key: SUA_CHAVE_AQUI"
```

| Status | O que fazer |
|--------|-------------|
| `IN_PROGRESS` | Aguarde e consulte novamente em 5–10s |
| `COMPLETED` | Use `generated[0].url` — **baixe imediatamente**, a URL expira |
| `FAILED` | Tente novamente com o mesmo prompt |

## Prompt padrão Zavvy

O prompt base completo e validado está em [Referência Visual → Prompt base reutilizável](/docs/referencia-visual#prompt-base-reutilizável). Cole-o no campo `prompt` da API.

---

<Callout type="tip">
**Prompts em inglês performam melhor** no Seedance. Os prompts da Zavvy já foram validados — se for adaptar, mantenha os termos técnicos em inglês (morphsuit, bokeh, depth of field, cinematic lighting).
</Callout>

<Callout type="warning">
**URL temporária:** o link do vídeo em `generated[0].url` expira. Faça o download imediatamente após o status `COMPLETED`.
</Callout>
