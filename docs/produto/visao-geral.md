# Produto

**Zavvy** é uma plataforma de agendamento para profissionais autônomos com confirmação automática via WhatsApp.

## Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| Link de agendamento | URL pública para clientes agendarem sem criar conta |
| Confirmação via WhatsApp | Mensagem automática enviada ao cliente ao agendar |
| Lembrete 24h antes | Mensagem automática no dia anterior ao horário |
| Agenda | Visualização por dia, semana ou lista |
| Gestão de clientes | Histórico, mensagens e anotações por cliente |
| Equipe | Múltiplos profissionais sob o mesmo negócio |
| Assistente IA | Responde clientes automaticamente pelo WhatsApp |
| QR Code | Alternativa ao link para compartilhamento físico |
